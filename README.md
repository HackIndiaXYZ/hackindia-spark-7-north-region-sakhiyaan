
  # Rural Women's Empowerment App

  This is a code bundle for Rural Women's Empowerment App. The original project is available at https://www.figma.com/design/CpoQ1IQy9I6LOqjN7nJ2nF/Rural-Women-s-Empowerment-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server with hot reloading enabled.

  The app will be available on `http://localhost:5173`, and changes to files in `src/` should refresh automatically.
